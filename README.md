# MatchingServer

Matching server for Magica Duel  
Team project of Ajou Univ. Media Software Engineering Lecture

## Includes

-   Node.js
-   Socket.io
-   MySQL

## Installation

Yarn package manager recommended

```
npm install -g yarn
```

Install project dependencies

```
yarn install
```

## Usage

Run server

```
yarn start
```

Run test

```
yarn test
```
